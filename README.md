# QR Code Generator

A simple Python GUI application to generate QR codes from text or URLs.

## Requirements

- Python 3.x
- qrcode library with PIL (install with `pip install qrcode[pil]`)
- tkinter (included with Python)

## Usage

Run the application:

```bash
python main.py
```

Enter the text or URL in the input field, click "Generate QR Code" to create and display the QR code. Use "Save QR Code" to save it as a PNG file.
